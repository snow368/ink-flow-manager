# 偏门小众 niche 高利润 Agent 系统

## 核心原则
- 成本: ¥50-300 人民币
- 售价: €80-600 欧元
- 毛利率: 75-88%
- 量: 偏门小众，不碰大类
- 利润率优先，量不优先

## Niche 选择标准
- Amazon 搜索结果 < 5000 商品
- 头部竞品差评率 > 15%
- 月搜索量 1000-10000
- 竞品平均评论数 < 500
- 有明确的目标人群和文化属性

## 6 个发现方向
1. 竞品逆向 - 找对手没做好的东西
2. 用户痛点 - 从 Reddit/评论里挖抱怨
3. 供应链上游 - 1688 公模改私模
4. 趋势预判 - Google Trends 上升但竞品少
5. 内容社区 - YouTube/TikTok 高互动但无对应产品
6. 跨界迁移 - A 品类成熟方案迁移到 B 品类

## 验证 5 关
1. 工程可行性 - 工艺成熟，不良率可控
2. 利润核算 - 净利 > 20%
3. 竞争验证 - 竞品质量差，有差异化空间
4. 需求验证 - 痛点真实，愿意付高价
5. 外观改良 - 3 个方案，溢价覆盖成本
