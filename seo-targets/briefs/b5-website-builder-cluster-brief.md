## 1. 独家角度（Exclusive Angle）

> 本簇为 **MOFU/BOFU 商业簇**，角度形态 = 最清晰的结构 + 最可信的比较 + 最强信任信号，**不硬编 POV**。

竞品 Top 页现状：
- **Wix / Squarespace / Shopify**：通用建站器，套"tattoo template"壳，但不懂纹身店——无内置预约、无作品集管理、无定金/客户 CRM，用户要自己拼插件。
- **Linktree / Lnk.Bio / Beacons**：本质是 **link-in-bio**（单页一堆链接），不是真网站，Google 当单页收录、SEO 极弱。
- **Tattoo Studio Pro / Format / Pixpa / Snapps.ai**：垂直 SaaS，做得比上面懂行，但多为付费、且各自只讲一个切片（portfolio / builder / 对比）。

我们的差异点（吃下全簇）：
**「一个为纹身店而生、开箱即用的真网站——预约 + 作品集 + 客户管理内置，免费起步，上线即被 Google 收录，Pro 可绑工作室自己的独立域名。」**
- vs 通用建站器：不用学拖拽、不用拼插件，纹身店场景开箱即用。
- vs Linktree：是真·独立站（每店独立 URL、独立 meta、可被 Google 当独立页面收录），不是 bio 单页。
- vs 多数建站器（只给 `xxx.platform.com` 子域）：Pro 绑的是**工作室自己的独立域名**（如 `yourstudio.com`），**不是子域**——子域站 SEO 权重与品牌信任弱于独立域名，这是硬差异。
- **信任信号（第一手）**：InkFlow 托管的纹身店站点**已被 Google 实际收录**（用户确认），且 `ink-flows.com` 本身是**独立域名（非子域）**——竞品 link-in-bio / 子域站做不到的硬差异；本簇每页用"live example / indexed on your own domain"做信任锚。

与 InkFlow 转化衔接：免费网站 → 引导注册 → Pro 解锁**自有独立域名**/高级模板/去品牌。
