## 适用分类
中文出海圈, 内容站策略

## 核心策略：低 KD + 多内页聚合

> "聚沙成塔" — 选竞争很小的词，大量做内页，单个页面流量小但总量大。

**典型案例：** picturethisai 有 500 万页面，单页几百访问，聚合后月 150 万访问，72.3% 来自 SEO。

### 对 InkFlow 的启发

InkFlow 不走 500 万页这种量级（那是图片识别类网站），但核心逻辑可以用：
- **Feature 页面**：每个功能单独一个 Landing Page（Booking、POS、CRM、Waiver、Inventory...）
- **城市页（后期）**：后期每个城市一套页面，"Tattoo Studio Software in [City]"
- **对比页系列**：每个竞品一张对比页，按意图分组优化

每个页面流量不大，但合起来可观。

## 他的模式 vs 我们的模式（核心区别）

| | hezhiyan7 | InkFlow |
|--|-----------|---------|
| **模式** | 内容站 / 广告联盟 | 产品 SaaS |
| **Niche** | 不固定，追热词 | 固定：纹身工作室管理 |
| **内容量** | 500 ~ 5,000,000 页 | 前期 20-50 页精品 |
| **转化** | 广告点击 / Affiliate | 免费注册 → 付费订阅 |
| **核心能力** | 发现新词 → 快速上站 | 渗透 niche → 建立权威 |

他的模式：**发现热词 → 自动上内容 → 吃流量 → 广告变现**
我们的模式：**niche 已定 → 找低竞争入口 → 高质量渗透 → 转化付费**

我们不需要"发现新词"，我们需要的是**在现有 niche 里找到 KD 低的切入角度**：

| 高竞争（难做） | 低竞争（我们的机会） |
|----------------|-------------------|
| "tattoo studio software" | "tattoo shop aftercare email automation" |
| "tattoo booking system" | "multi-artist commission tracking" |
| "tattoo POS" | "deposit based booking tattoo software" |
| "tattoo CRM" | "digital tattoo consent form app" |

同一个 niche，换角度，竞争完全不同。

## 找词方法论（只取对我们有用的）

### 1. 不要只看 KD 值
多个关键词同一意图要分组优化，吃下所有相关长尾。

### 2. 信息溯源（核心技能）
通过追踪热点传播源头，**提前发现即将爆发的关键词**，抢在同行之前上站。

来源包括：Reddit 板块、大博主、大平台、行业新闻。

### 3. 词根找需求
通过词根扩展找到用户真实需求。

### 对 InkFlow 的启发（修正版）

他这套适合"不知道做什么就追热词"的人。**我们不需要。**

我们要的是：
1. 列出 niche 内所有关键词（纹身工作室管理）
2. 用工具查 KD，找到 < 20 的入口
3. 分组按意图（商业/交易/信息）
4. 先做商业/交易意图的（转化高）
5. 每个词产出一篇高质量内容

而不是：
1. 去 Reddit 找新热词
2. 快速上站抢排名
3. 吃一波流量走人

## 转化链路

```
流量获取 → 首页引导 → 登录注册 → 首次体验 → 付费升级 → 留存活跃
```

SaaS 每个环节都要看数据、A/B 测试。

## 个人判断（最终版）

### 从他身上学到的最重要一课
**不是找词方法论，而是"在固定 niche 里找到低竞争入口"的思路。** 这一课可以用在任何垂直 SaaS 的 SEO 上。

### 适合 InkFlow 的：
- ✅ 低 KD 切入思路（在纹身软件 niche 内找竞争低的词）
- ✅ 同一意图的关键词分组优化
- ✅ SaaS 转化链路框架（通用）
- ✅ **渐进结果展示变现（2026/06/03 新文章）** — 免费工具 → 看到部分结果 → 付费解锁完整结果。详见 [[hezhiyan7-progressive-result-display]].md

### 不适合的：
- ❌ 「发现新词追热点」—— 我们 niche 固定，不需要
- ❌ 「堆量几百几千页」—— 质量 SaaS 不适合
- ❌ 灰帽手法（如程序化批量内容）—— 风险高

### 冲突点判断
- **他说：** 每页几百访问就够了，靠量取胜
- **我说：** 对 InkFlow 来说，每页几百访问但 15% 转化率 > 每页几千访问但 0.5% 转化。我们追求的是**精准流量 + 高转化**，不是流量总量。
