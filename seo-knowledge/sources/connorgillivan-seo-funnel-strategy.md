## 适用分类
B2B SaaS SEO

## 核心框架：SEO Funnel Priority

### 漏斗三层

```
TOFU（顶部）— 建立认知
  → 用户在学一个话题，没有购买意图
  → 教育类内容
  → 最受 Google AI Overview 影响
  → 例："what is link building in seo?"

MOFU（中部）— 研究与比较
  → 用户在比较选项
  → 对比文、How-to、案例研究
  → 例："best link building tools in 2025"

BOFU（底部）— 准备购买
  → 用户在找解决方案
  → 高意图、直接转化
  → 例："link building agency detroit"
```

### 执行策略

```
🥇 BOFU 先做，做到没词可做
🥈 然后 MOFU，填充漏斗中部
🥉 TOFU 留给 AI（不优先投入）
每月至少 5 篇内容
外链集中建到 BOFU 页面
```

---

## 关键原则

### 1. BOFU First Until You're Out
- 高意图词有限，先全部覆盖
- 这些词直接产生收入
- 每篇 BOFU 内容都是"收入页面"

### 2. 然后 MOFU Fill the Funnel
- BOFU 词做完了，做 MOFU
- MOFU 内容把用户从研究阶段推到购买阶段
- 对比文、Best-of 列表

### 3. 把 TOFU 留给 AI
- TOFU 内容最容易被 AI Overview 吃掉流量
- Connor 的建议：**不优先投入 TOFU**
- 让 AI 去覆盖教育类内容，你做转化类

### 4. 每月 5+ 篇内容
- 稳定的内容产出节奏
- 每篇瞄准一个关键词意图层级

### 5. 外链集中到 BOFU
- 不分散外链权重
- 所有链接建设集中打 BOFU 页面
- BOFU 页面转化率最高，外链投资回报最大

---

## BOFU vs MOFU vs TOFU 优先级决策

| 维度 | TOFU | MOFU | BOFU |
|------|------|------|------|
| 搜索量 | 高 | 中 | 低 |
| 转化率 | 极低 | 中 | 高 |
| AI Overview 风险 | 高 | 中 | 低 |
| 外链 ROI | 低 | 中 | 高 |
| **优先级** | **3** | **2** | **1** |

## 对 InkFlow 的启发

### 映射到 InkFlow 的关键词

**BOFU（先做）：**
- "tattoo studio software"（交易意图）
- "tattoo booking system for artists"
- "digital waiver app for tattoo shops"
- "InkFlow Pro pricing"（品牌词）
- "tattoo studio CRM"
- 所有 vs 竞品页（vs TSP, vs Booksy, etc.）

**MOFU（第二波）：**
- "best tattoo studio software 2025"
- "how to choose tattoo studio software"
- "tattoo shop management software features"
- "tattoo software with payment processing"

**TOFU（留给 AI，不优先）：**
- "what is a tattoo waiver"
- "how do tattoo studios work"
- "tattoo aftercare tips"
- 这些已经有博客覆盖，但不需要额外投入
