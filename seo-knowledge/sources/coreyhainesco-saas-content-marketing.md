## 核心方法论

### 1. 五层认知框架（Five Stages of Awareness）

基于 Eugene Schwartz 的经典框架，Corey 将其应用于内容营销：

| 层级 | 状态 | 内容策略 |
|------|------|---------|
| **Unaware** | 不知道问题存在 | 教育内容、痛点唤醒 |
| **Problem Aware** | 知道问题，不知道方案 | 介绍方案类型 |
| **Solution Aware** | 知道方案，不知道你的产品 | 对比、功能展示 |
| **Product Aware** | 知道你的产品，在评估 | 案例、评价、Demo |
| **Fully Aware** | 准备购买 | 定价、试用、CTA |

**关键：** 内容类型不能死板绑定到漏斗层级。Webinar 可以是 TOFU，博客可以是 BOFU。

### 2. 内容复用方法论（Atomization）

> "Everything's a remix."

一份核心内容 → 拆散到多平台：

```
一篇深度研究/拆解
  → Newsletter 节选
  → Twitter 线程
  → 短视频片段
  → 播客单集
  → LinkedIn 帖子
```

### 3. 自有平台优先

| 平台类型 | 例子 | 策略 |
|---------|------|------|
| 🟢 **Owned** | 网站、博客、Newsletter、播客 | **重点投入** |
| 🟡 **Rented** | Medium、LinkedIn | 分发，不依赖 |
| 🔵 **Borrowed** | Twitter 互动 | 建立人设，不依赖 |

### 4. 新 SaaS 营销人员的"头 90 天"

- **第 1-30 天**：纯客户和产品研究——做客服、录 Demo 视频、读知识库、跟客户聊天。**不做任何改动。**
- 系统收集反馈 → 识别模式 → 建立产品改进依据

### 5. 避免算法依赖

> 基于第一性原理、白帽策略和营销心理学（钩子、文案、风险逆转、紧迫感）构建基逻辑的营销策略。过度依赖单一算法或渠道是危险的。

## 对 InkFlow 的启发

### 五层认知框架 → 映射到我们的内容

| 层级 | InkFlow 内容 |
|------|-------------|
| Unaware | "纹身店管理为什么需要软件" |
| Problem Aware | "如何减少 No-Show"、"Waiver 数字化指南" |
| Solution Aware | "免费 Waiver Generator" |
| Product Aware | "InkFlow vs TSP 对比" |
| Fully Aware | 免费试用 + Pricing 页 |

### 内容复用

一篇 "How to Digitize Tattoo Waivers" 博客可以：
- Newsletter 节选
- Twitter 线程
- LinkedIn 帖子
- 免费工具（Waiver Generator）的配套内容

### 自有平台优先

我们的 Astro 站 + Newsletter + 免费工具都是自有平台，不依赖社媒。社媒只是分发渠道。
