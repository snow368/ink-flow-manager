# Alt Text 不是描述，是关键词机会

**来源：** 用户直接提供
**类型：** 🟢 图片 SEO / Accessibility

---

## 核心

> "Alt text is NOT a description. It's a keyword opportunity."

| ❌ 错误 | ✅ 正确 |
|---------|---------|
| "a pair of shoes" | "women's red Nike running shoes size 7" |
| "tattoo machine" | "Cheyenne Hawk rotary tattoo machine with soft grip" |
| "client chart" | "tattoo studio client intake form with design sketch" |

**为什么：** Screen readers + Google 都读 alt text。一个字段同时服务 accessibility 和 rankings。

---

## 对 InkFlow 的应用

网站上的每张图片 alt text 按这个标准写：

| 图片类型 | Alt text 公式 | 例子 |
|---------|--------------|------|
| 产品截图 | [产品名] + [功能名] + [场景] | "InkFlow client management dashboard showing appointment schedule" |
| 纹身作品 | [风格] + [部位] + [工作室] | "black and grey realism forearm tattoo at InkFlow studio" |
| 团队/工作室 | [人名] + [角色] + [动作] | "tattoo artist Sarah using InkFlow scheduling on iPad" |
| Logo | 品牌名 + "logo" | "InkFlow tattoo studio software logo" |
