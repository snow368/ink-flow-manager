## 战术拆解（作者 4 点 + 本 KB 视角注释）

| # | 作者原话 | 实操含义 | 本 KB 对应 |
|---|---------|---------|-----------|
| 1 | Change your file name | 文件名含关键词（如 `blue-floral-tattoo-design.jpg`，非 `IMG_4821.jpg`） | 通用 SEO 基础；seo-b2c 图片 SEO 章可补 |
| 2 | Add a keyword to the ALT text | alt 用描述性 + 关键词（非堆砌） | 无障碍 + Image SEO 双重收益 |
| 3 | Reduce image size by compressing | 压缩降体积 → 提速 → CWV/LCP 改善 | 与 seo-speed-optimizer / CWV 相关 |
| 4 | Add ImageObject schema | 页内 JSON-LD 标 `ImageObject`（contentUrl/name/caption/license） | 🚨 本 KB 四套 per-type skill 均未单独覆盖 `ImageObject`；待补 |

## 关键数据（SELF-REPORTED，待验证）

- 帖子 A：单站 Image SEO 带来 **9,200+** 访客
