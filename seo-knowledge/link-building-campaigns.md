> 写给谁看：SEO 负责人、外链专员
> 目标：从 0 到 1 建立可衡量的外链战役体系
> 最后更新：2026-07-13

---

## 一、外链运营总逻辑

```
外链不是一个"任务"，是一个"战役"。
每次战役由 4 个阶段组成：研究 → 执行 → 跟进 → 复盘
一个战役周期 = 4-6 周
```

**核心原则**:
1. **质量 > 数量**：1 条 DA 60 的相关链接 > 10 条 DA 30 的通用链接
2. **速度自然**：每月 10-20 个新引用域名看起来自然；200 → 0 看起来很假
3. **锚文本多样性**：40% 品牌锚 / 25% 裸 URL / 20% 通用 / 10% 部分匹配 / **5% 精确匹配**
4. **先有内容资产，再发出 outreach**：没有可链接的内容 = 空手套白狼

## 二、外链战役执行框架（4 周/战役）

### 战役前：内容资产准备（1 周）

**哪些内容值得被链接？**（按可链接性排序）

```
1️⃣ 原创数据/行业报告 → 最高（天然链接磁石）
2️⃣ 免费工具/计算器 → 高（实用性强）
3️⃣ 深度指南/终极帖  → 中（需要推广）
4️⃣ 对比/替代页     → 中（商业意图）
5️⃣ 普通博客文章     → 低（几乎无人主动链接）
```

**规则**: 在发起 outreach 之前，确保有一篇"值得链接"的内容作为目标页。否则先把内容资产做好。

### 第 1 周：研究与建表

```
Day 1-2: 发现候选网站
  ├→ 竞品外链反查（Ahrefs/SEMrush → 竞品域名 → 外链报告）
  ├→ "best [关键词]" 搜索结果 → Top10 页面
  ├→ 行业目录 / 协会网站列表
  └→ 工具：Ahrefs Site Explorer / SEMrush Backlink Analytics

Day 3-4: 筛选与排序
  ├→ 筛选条件：DA ≥ 30 + 内容相关 + 有真实编辑团队
  ├→ 标记分类：
       Type A: 高价值（DA 60+, 强相关）→ 定制 pitch
       Type B: 中等（DA 30-60, 一般相关）→ 模板 pitch
       Type C: 目录/列表（DA 30+）→ 快速提交
  └→ 目标：每个战役 30-50 个候选

Day 5: 制定 Pitch 策略
  ├→ Type A: 用 Skyscraper 技术（已经链接到竞品的页面 → 展示我们更好的内容）
  ├→ Type B: 用 Broken Link Building（找到 404 外链 → 推荐自己的可替代内容）
  └→ Type C: 直接提交或目录申请
```

### 第 2 周：Outreach 执行

```
Day 1-3: Type A 定制邮件（5-10 封/天）
  ├→ 每封个性化（提及对方的一篇文章 + 为什么我们的内容补充了它）
  ├→ 标题要抓眼球但诚实："We updated [topic] — you linked to the old version"
  └→ 200 字以内，可扫描

Day 4-5: Type B 批量邮件（15-20 封/天）
  ├→ 模板邮件 + 少量个性化（替换名字/网站名）
  ├→ 主题行: "Quick suggestion for [网站名]"
  └→ 发送工具: Postaga / BuzzStream / Hunter.io

Week 2 全程: 记录跟进
  ├→ 已发送 | 已回复 | 已同意 | 已拒绝 | 已沉默
  └→ 7 天后追一次（"Just following up on my email below…"）
```

### 第 3 周：跟进

```
Day 1: 追上周一发的未回复邮件
Day 3: 追上周三发的未回复邮件
Day 5: 追上周五发的未回复邮件
全程: 回复任何同意的 → 确认链接已上线
```

### 第 4 周：复盘

```
指标:
  ├→ 发送量: ____
  ├→ 回复率: ____ （基准线 10-20%）
  ├→ 接受率: ____ （基准线 5-15%）
  ├→ 已获得链接数: ____
  └→ 平均 DA: ____

质量判断:
  获得的链接是否在相关内容页上？
  锚文本是否自然？
  链接是否 dofollow？
  页面是否有真实流量？

下战役调整:
  回复率 < 10% → 改进你的 Pitch（太长？不够相关？）
  接受率 < 5% → 改进你的内容资产（不够值得链接？）
  平均 DA < 30 → 选更高价值的目标
```

## 三、五种核心外链战术的操作流程

### 战术 1: Broken Link Building（破链法）

**逻辑**: 找到目标网站上指向 404 页面的外链 → 用你的内容替换它。

```
Step 1: 在竞品外链中找 404（Ahrefs → 404 error 过滤）
Step 2: 找到匹配的内容或创造一篇
Step 3: 发邮件告诉对方"你链到一个不存在页面了，我这有更好的替代"
Step 4: 对方获得：修复了一个用户体验问题；你获得：一条相关外链
```

**成功率**: 中等（8-12%），但链接质量高

### 战术 2: Skyscraper（天塔法）

**逻辑**: 找到已有很多外链的内容 → 做得更好（更长 + 更新 + 更好看）→ 告诉链接到它的人

```
Step 1: 找一篇已有大量外链的竞品内容（Ahrefs → Best by Links）
Step 2: 做得更好：
  ├→ 更新数据（2026 年 vs 2020 年）
  ├→ 加表格 + 可视化
  ├→ 加原创数据点
  └→ 加 FAQ Schema
Step 3: 列出所有链接到原始页面的网站
Step 4: 逐个 outreach："我们发布了 2026 年更新版"
```

**成功率**: 中等偏高（15-25%）

### 战术 3: HARO / Featured.com（记者外链）

**逻辑**: 记者需要专家意见 → 你提供 → 记者引用你 + 链接

```
每月预算：免费版够用
操作流程：
  Day 1-3: 每天查看 HARO 邮件，筛选相关主题
  Day 4-7: 快速响应（前 6 小时提交转化率高 20%）
  响应规则：
    ├→ 200 字以内
    ├→ 要有"可直接引用"的句子
    ├→ 含自己网站链接
    └→ 展示专业身份

Expected: 5-15% 成功率
```

**来源**: [OutreachDesk — HARO Guide](https://outreachdesk.com/haro-link-building)

### 战术 4: #JournoRequest（X/Twitter 记者外链）

**逻辑**: 记者在 X/Twitter 上发求助 → 你回复 → 被引用

```
搜索: "#JournoRequest [topic]" 或 "#JournoRequest [industry]"
工具: 用 JournoFinder / IFTTT 设置提醒
响应: 在 Twitter 上直接回复（约 70% 的国家级媒体曝光来自记者直接请求）
```

### 战术 5: 数字 PR / 原创数据（季度战役）

**逻辑**: 发布有新闻价值的数据 → 媒体自然报道 → 天然外链

```
成本: $500-2000/场（工具 + 设计）
周期: 1-2 月准备 + 1 月推广
案例: 网络研讨会平均每条 140+ 外链
成功关键:
  ├→ 独特角度（不"101"，而是"2026 Tattoo Industry Data: 70% of Studios..."）
  ├→ 联合知名主持人
  ├→ 原创数据
  └→ 多渠道分发
```

**来源**: [Noel Ceta — Webinar 外链策略](https://x.com/noelcetaSEO/status/2075286588276257122)

## 四、外链质量评估体系

| 特征 | 高质量 | 低质量 |
|------|--------|--------|
| DA/DR | ≥ 40 | < 20 |
| 内容相关性 | 紧密相关 | 完全不相关 |
| 链接位置 | 正文 | 侧边栏/页脚 |
| 锚文本 | 品牌/裸 URL | 精确匹配关键词 |
| 页面流量 | 月 ≥ 1000 次搜索 | 月 < 100 |
| 是否编辑链接 | 编辑主动添加 | 付费/互惠 |
| 是否 dofollow | 是 | 否（nofollow 也有效，但权重低） |

### ⚠️ 目录站评估：不要只看 DR

> **提交成功 ≠ 获得外链。** 页面存在 ≠ Google 收录。很多 DR 60+ 的目录站自然流量很低、产品页根本没被 Google 收录。— 唐先僧 (来源 066)

**增加两个被忽视的指标**：

| 新增指标 | 怎么查 | 高质量标准 |
|---------|--------|-----------|
| **Google 收录速度** | 提交后 site: 检查 | 48 小时内收录 |
| **页面收录率** | 抽 10 个已提交页面查 site: | ≥ 80% 已收录 |

**操作标准**：
```
目录站提交 → 2 周后 site: 检查产品页面
  已收录 → 继续合作（这个目录出工出力）
  未收录 → 放弃，不要继续浪费提交时间
  6 个月未收录 → 之前投的时间全浪费了
```

> DR 可以被互投刷高，不代表有真实收录能力。优先选收录快的目录（通常 = 有真实编辑团队）。

**可接受的 nofollow 链接**:
- 社媒 profile 链接（nofollow 但加速品牌搜索）
- 博客评论（自然头像）
- 行业目录（流量价值 > SEO 价值）

## 五、常见的过量/危险信号

```
精确匹配锚文本占比 > 10-15% → 触犯 Penguin
月外链增速 > 50/月 → 触犯人工审查
30%+ 外链来自相同 IP/C 段 → 不自然
> 20% 的链接使用相同锚文本 → 不自然
链接来自无关网站（如 casino/seo 论坛）→ 有害
链接突然爆发后归零 → 看起来像购买了链接
```

## 六、外链战役年度节奏

```
Q1: 基础建设
  ├→ 目录/列表提交（20+ 条）
  ├→ Type A/B outreach（15-20 条）
  └→ 内容资产准备（行业报告 1 份）

Q2: 加速
  ├→ HARO 月度参与
  ├→ 嘉宾文章 2 篇
  └→ Broken Link 战役

Q3: 规模
  ├→ 原创数据发布（数字 PR 战役）
  ├→ 网络研讨会 1 场
  └→ Skyscraper 战役

Q4: 收割
  ├→ Q1-Q3 战役的尾流（链接还在自然出现）
  ├→ 内容修剪 + 重推
  └→ 下年度规划
```

## 七、按业务类型的外链战役策略

### 目标网站来源差异

| | SaaS | B2C 电商 | B2B |
|--|------|---------|-----|
| **#1 来源** | G2/Capterra + SaaS 目录 | 评测站/导购站 | 行业协会/商会 |
| **#2 来源** | 集成伙伴目录 | Influencer 合作 | 商业平台(Alibaba/ThomasNet) |
| **#3 来源** | Guest Post (SaaS 博客) | 社媒(IG/TikTok) | 认证机构/资质目录 |
| **#4 来源** | Product Hunt / 工具聚合页 | Pinterest(视觉) | Dun&Bradstreet/Kompass |
| **#5 来源** | 开发者目录/API 目录 | Instagram 购物 | 行业媒体/白皮书平台 |

### Outreach 对象差异

| | SaaS | B2C 电商 | B2B |
|--|------|---------|-----|
| **对象** | SaaS 博主、Capterra 编辑、集成伙伴 | 美妆/生活方式博主、导购站编辑 | 行业协会编辑、行业媒体记者 |
| **Pitch 角度** | "帮助你读者的 SaaS 推荐" | "本季度最受欢迎的 X 产品" | "行业最新数据/白皮书" |
| **内容资产** | 免费工具、对比页、行业报告 | 产品评测、购买指南、使用场景 | 白皮书、案例研究、ROI 计算器 |
| **HARO 话题** | SAAS/技术/创业 | 生活方式/美妆/时尚 | 行业趋势/采购/供应链 |

### Schema 类型差异（影响 AI 引用的外链信号）

| | SaaS | B2C 电商 | B2B |
|--|------|---------|-----|
| **外链目标页 Schema** | SoftwareApplication | Product + Offer + Review | Organization + Product |
| **推荐目录 Schema** | SoftwareApplication | WebSite | Organization |

## 八、AI 自动化外链（新路线）

> 手动战役（前七节）是"精耕细作"，本节是"规模分发"——两者互补，不可替代。

### 适用时机

| 阶段 | 适合用什么 |
|------|-----------|
| 新站/0-500 词排名 | AI 自动化分发打基础（品牌提及 + 跨平台存在感） |
| 已有排名/流量 | 手动战役做高质量编辑外链 |
| 有原创数据/工具 | 数字 PR + 手动 Outreach 双线并行 |

### 核心方案：n8n + Claude 外链引擎

详见 `sources/071-claude-n8n-backlink-automation.md`，核心逻辑：

```
Google Sheet（主题队列）
    → Claude 生成 4 版独特内容（WP/LinkedIn/Reddit/Dev.to）
    → 自动发布到各平台（含主站链接 + 交叉链接）
    → 更新追踪表
```

**成本**：~€3/月（Claude API），n8n free tier 够用。

### 2026 AI 可见性新规则

| 旧规则 | 新规则 |
|--------|--------|
| 只要 dofollow | nofollow ≈ dofollow（AI 引用权重） |
| 孤立的单点外链 | 跨平台存在感更重要 |
| DA/DR 决定权重 | 品牌搜索量 = 最强 AI 引用预测 |
| 链接数量 = 排名 | 4+ 平台提及 → AI 引用率 +2.8x |

### InkFlow 适配路线

```
Phase 1: 先用 n8n 把现有 81 页内容做跨平台分发（Tattoo forum/Medium/Dev.to）
Phase 2: Claude 批量化写 Guest Post + HARO 响应
Phase 3: 建 Web 2.0 集群（各平台交叉链接 → 权重回流主站）
Phase 4: 等 GSC 有排名数据 → 手动战役收割高质量编辑外链
```
