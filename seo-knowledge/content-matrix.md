## 二、搜索意图分类（3类）

来源：微信推送「SEO完整框架拆解」(2026-07-10)

| 类型 | 用户行为 | 转化潜力 | 竞争度 | 对策 |
|------|---------|---------|--------|------|
| **购买词** | 已经决定买在比较选项。如"best trail running shoes 2025" | 最高 | 最激烈 | 对比页 + 工具榜单 |
| **痛点词** | 遇到问题在找方案。如"why shoes wear out fast" | 中等 | 低 | 教程页 + 指南博客 |
| **场景词** | 特定场景搜索。如"shoes for marathon training" | 高 | 中等 | 场景对比页 + 选购指南 |

**核心数据：** Ahrefs 2024年10亿页面分析：排名前20%页面中 **64.7%流量来自月搜索量低于100的长尾词**。

## 四、内容生产流程（10步）

```
① 选题 → 关键词矩阵中找KD 0-15的词
② 搜SERP → 看首页排什么内容格式
③ 找差距 → 竞品没写的角度（信息增益）
④ 定格式 → 对照上表选内容格式
⑤ 写H1-H3 → 按关键词分组规划
⑥ 写内容 → 首段BLUF → 段落自包含 → 列表/表格
⑦ 写FAQ → 3-5个真实问题
⑧ 加内链 → 至少3条
⑨ 加Schema → 按类型
⑩ 加CTA → 注册/试用/了解更多
```

## 七、先给答案的内容结构

来源：微信推送「SEO完整框架拆解」(2026-07-10)

**传统结构（不要用）：** 开头铺垫背景 → 中间分析展开 → 结论放最后 → 跳出率高

**答案优先结构（推荐）：**
```
第一段 → 直接给核心答案
→ H2/H3 → 展开详细分析
→ 补充细节、场景、案例
→ 留存和转化更高
```

原理：Google 算法倾向在搜索结果顶部直接给出答案（Featured Snippet）。首段给出答案的页面更容易被提取为精选摘要。

## 九、程序化 SEO

来源：微信推送「SEO完整框架拆解」(2026-07-10)

**适用条件：** 已有50-100篇手写内容 + 一定 DA

**质量门槛：**
1. 数据独特性 — 加自己分析维度
2. 内容差异化 — 条件逻辑输出不同结论
3. 重复控制 — >80%相似度合并或用canonical
4. 人工审核 — 首批页面抽检

## 十、技术 SEO 检查清单（7项）

来源：微信推送「SEO完整框架拆解」(2026-07-10)

1. 收录与抓取 — GSC Coverage
2. XML Sitemap — 提交到 GSC
3. Canonical Tag — 防重复内容
4. 重定向链 — 避免 A→B→C
5. 速度 — LCP<2.5s, INP<200ms, CLS<0.1
6. 移动端 — 移动优先索引
7. HTTPS — 必须

## 十一、增长策略（4种）

来源：微信推送「SEO完整框架拆解」(2026-07-10)

1. **竞品替代** — 跟踪竞品排名，写更好版本抢占
2. **非品牌增长** — 非品牌词占自然流量 >60%
3. **程序化增长** — 程序化+集群可复制引擎
4. **复合增长** — SEO+社媒+邮件飞轮

## 十二、新手常见错误

来源：微信推送「SEO完整框架拆解」(2026-07-10)

1. 只追搜索量大的词
2. 内容没结构（无集群）
3. 忽略技术 SEO
4. 期望短期见效（需3-6个月）
5. 只做SEO不做多渠道
