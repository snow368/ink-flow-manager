## 本地语音 SEO 四要素

### 1. LocalBusiness Schema
- 名称、结构化地址
- E.164 格式电话
- 营业时间
- 经纬度、服务区域
- Google Maps 链接

### 2. NAP 一致性
跨 GBP、Bing Places、Apple Maps、Yell、Facebook 等完全一致。不一致 = 语音结果信心降低。

### 3. 对话式内容结构
- 用**问题式标题**: "What web design services do you offer?" 而非 "Our Services"
- 每个问题后紧跟 **40-60 字** 直接答案
- 每服务页应有 **8-15 个 FAQ 条目**，覆盖费用、时间线、流程、服务区域、资质和常见顾虑

来源: [DigitalMarketingExpertTeam — Conversational Search 2026](https://digitalmarketingexpertteam.com/conversational-search-local-business)

### 4. 自然融入本地限定词
- ✅ "Our web design studio in Northlew, Devon" 优于 "our studio"

## Schema 要求 (语音搜索)

| 类型 | 用途 |
|------|------|
| **LocalBusiness** | 核心企业信息 |
| **FAQPage** | 常见问题语音回复 |
| **Speakable** | 语音专用标记 |
| **HowTo** | 操作指南类查询 |

## 语音搜索四类意图

| 类型 | 示例 | 优化重点 |
|------|------|---------|
| 信息型 | "How to remove a tattoo?" | 直接答案 40-60 字 |
| 导航型 | "Where is the nearest tattoo shop?" | GBP + LocalBusiness |
| 交易型 | "Order tattoo aftercare cream" | Product + Offer Schema |
| 本地型 | "Best tattoo artist near me" | LocalBusiness + GBP |

## 30 天语音搜索行动计划
1. **第 1 周**: 审计 + 竞品 FAQ 研究
2. **第 2 周**: 以 40-60 字 H2 格式创建 FAQ 内容
3. **第 3 周**: 部署 LocalBusiness + FAQPage + Speakable Schema
4. **第 4 周**: GBP 全面优化
