## 🏭 行业协会（高价值白帽外链）

行业协会的 `.org` 链接 Google 权重很高：

| 类型 | 找法 |
|------|------|
| 本地商会 | 搜 "城市名 + chamber of commerce" |
| 行业协会 | 搜 "行业 + association" |
| 专业认证机构 | 搜 "行业 + certification" |
| 校友会 | 毕业院校目录 |

**操作：** 如果你已经是会员但没被列在目录里，发邮件给管理员要求加上。

## 执行顺序

```
第 1 周：Google Business Profile + LinkedIn + Crunchbase + Trustpilot
第 2 周：Dun & Bradstreet + BBB + 当地商会
第 3 周：行业特定目录（Kompass、ThomasNet 等）
第 4 周：Clutch / GoodFirms / UpCity（服务商用）
持续：行业协会 + 会员目录
```
