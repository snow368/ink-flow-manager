# SEO 每页执行标准

---

## 适用分类
内容策略, 每页执行

## 每页必须做的 10 件事

写每个页面时，按这个顺序做：

### 1. 定目标关键词
从关键词表选这个词页面的核心词。
```
例：/features/waivers → 目标词 "digital tattoo consent form app"（KD 5-10）
```

### 2. 写 Title（≤60字符）
格式：`{目标关键词} — {品牌名} | {补充}`
```
✅ "Digital Tattoo Consent Form App — InkFlow"
   tattoo consent form app（目标词） + InkFlow（品牌）
   48 chars ✓
```

### 3. 写 Description（≤160字符）
格式：`{价值主张} + {具体功能/数据} + {CTA}`
```
✅ "Go paperless with a digital tattoo consent form app. 
    E-signatures, QR check-in, medical history tracking. Start free."
    154 chars ✓
```

### 4. 写 URL
格式：`/{类别}/{关键词简写}`
```
✅ /features/booking
✅ /blog/how-to-digitize-tattoo-waivers
❌ /page?id=123&cat=booking
```

### 5. 写 H1
H1 必须含目标关键词，且跟 Title 不同。
```
Title: "Digital Tattoo Consent Form App — InkFlow"
H1:   "Digital Waivers for Tattoo Studios"
不同的写法，但都含 "digital" 和 waiver/consent 相关词
```

### 6. 正文自然嵌入关键词
位置：首段、H2、H3、FAQ
```
首段： "InkFlow's digital tattoo consent form app lets clients sign on their phone..."
H2:    "What to Look For in a Digital Waiver App"
H3:    "Medical History Tracking"
FAQ:   "Q: Is a digital tattoo consent form legally binding?"
```

### 7. 加内链（每页 3-6 条）
| 位置 | 链到 | 锚文本 |
|------|------|--------|
| 正文首次提 | 相关功能页 | "InkFlow's booking system" |
| CTA 区 | 注册页 | "Try InkFlow Free" |
| FAQ | 相关页 | 相关问题中自然链 |

### 8. 加 Schema
对应页面类型加结构化数据（SoftwareApplication / FAQPage / Article）

### 9. 加 OG + Twitter Card
确保分享到 X/LinkedIn 时有标题、描述、图片。

### 10. 发布后 IndexNow 推送
告诉搜索引擎有新内容了。

---

## 每月一次的整体检查

```
□ GSC 看哪些页面有排名、哪些没有
□ 排名下降的页面 → 更新内容+重新提交
□ 所有页面 Canonical 是否正确
□ 所有页面 Description 是否被 Google 重写了（如果被重写说明不够好）
□ 死链检查
□ 新关键词机会（AITDK 看竞品）
```
