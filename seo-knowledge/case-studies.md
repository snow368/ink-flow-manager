# SEO 实战案例库

> 真实项目验证过的策略和效果数据

## 案例一：新加坡B2B服务站 → 三个月55.6K点击

### 基础数据
| 指标 | 数值 |
|------|------|
| 市场 | 新加坡（600万人口） |
| 静默期 | 前6周几乎无数据 |
| 3个月总点击 | 55.6K |
| 3个月总曝光 | 352K |
| 平均CTR | 15.8%（优化前8.3%） |
| 平均排名 | 6.2 |
| 趋势 | 月增长加速（12K→18K→25K） |

### 关键策略

**1. 静默期生存（4-8周）**
- 不乱动网站结构（换URL/改Title/删页面会重置静默期）
- 持续做内容：已有核心页面做深做透
- 静默期不是做错了，是Google在评估

**2. 搜索意图匹配（核心杠杆）**
- Informational → 长文指南/FAQ
- Commercial → 对比页面
- Transactional → 服务落地页（定价+案例+CTA）
- 超过50%的搜索无点击是因为意图不匹配

**3. Meta优化（CTR 8.3%→15.8%）**
- Title: 主关键词靠前（前60字符）+ 本地化标识 + 差异化标签
- Description: 160字符内，含关键词+卖点+行动号召("Get a free audit")
- CTR提升→排名提升（自我强化循环）

**4. 内链审计**
- 修复三个问题：死胡同 / 重要页藏太深 / 锚文本不匹配
- 修复后收录一周增15%
- Topic Cluster模型：Pillar + Cluster Pages互相链接

**5. GSC复盘**
- 每周看Performance报告
- 找排名11-20的词（进前10点击量+300%）
- 找有曝光无点击的词（Meta优化）

## 案例二：匈牙利小语种独立站

### 基础数据
| 指标 | 数值 |
|------|------|
| 市场 | 匈牙利（900万人口） |
| 主关键词月搜索量 | 2400 |
| 长尾词 | 1.6K / 720 / 480 |
| 参考 | 另一个站月均750的主词，前三名每天1-5单 |

### 关键策略

**1. 建站注意事项**
- WordPress+WooCommerce
- 首页要丰富：品牌介绍+review+blog+FAQ
- H1只能1个，和Title关联但不相同
- 海报文字避免用H1标签

**2. 内容策略**
- Blog三类：Review / VS / Guide
- 每类可写1-2篇Pillar文章
- AI写文（2-5K字），结合经验补充
- 每天1-2篇，先发30篇左右
- 文章内链指向产品页

**3. 判断页面类型的方法**
搜Google前10名：
- 80%是文章 → 写Blog
- 80%是产品页 → 做产品页
- 80%是分类页 → 做分类页

## 共通规律
1. 小市场精准切入（不打大词打缝隙）
2. 都有4-8周静默期/1-3月排名波动
3. 都重视内链布局和持续内容输出
4. 长尾词转化率远高于泛词
