## 内容生产流程

### 我出初稿（每篇 30 分钟）
```
1. 按模板填充 H1-H3 结构
2. 写每段内容（结论前置）
3. 标注 [图片位置]
4. 标注 [链接:xxx]
5. 写 FAQ（GEO 优化）
```

### 你审（15 分钟）
```
1. 改行业术语、加真实案例
2. 截图补充
3. 核验功能描述
4. OK 了回复我
```

### 我改 + 发（15 分钟）
```
1. 处理反馈
2. 加图片
3. 发布到 Astro
4. IndexNow 推送
5. X link-in-reply 推广
6. LinkedIn 轮播图版本
```

### 内容发布后
```
Day 1: IndexNow 推送 + X 推广（link-in-reply）
Day 3: X 重新分享（不同切入点）
Day 7: LinkedIn 轮播图版本
Week 4: GSC 检查收录和排名
```
