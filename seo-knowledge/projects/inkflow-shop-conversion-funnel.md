# 城市页 → 纹身店转化漏斗

---

## 思路：免费 listing → 认领 → 管理 → 升级 Pro

跟 Google Business Profile、Yelp、Weedmaps 一样：

```
城市页展示店铺（免费）     →   引流
  ↓
"这是你的店吗？认领它"    →   免费注册
  ↓
认领后可以：编辑信息/加照片/看数据
  ↓
"想在线接单？升级到 Pro"  →   $29/月
```

## 城市页转化触点

每个店铺卡片上要加：

| 位置 | 文案 | 目标 |
|------|------|------|
| 店名旁边 | "Claim this listing" | 吸引店主点击 |
| 底部CTA | "Is this your studio? Manage your listing" | 认领入口 |
| 整个页面底部 | "Get listed on this page — Free" | 不在列表里的店 |
| 认领后 | 编辑信息/加照片/看浏览数据 | 免费价值 |
| 升级 | "Accept bookings directly → Start Free Trial" | Pro 转化 |

## 认领流程

```
点击 "Claim this listing"
  → 注册 InkFlow Free（$0）
    → 搜索自己店名 → 匹配到我们的数据
      → 认领成功 → 可编辑信息
        → 看店铺页面有多少浏览
          → "Upgrade to Pro to accept online bookings"
```

## 关键

不是一次性把这些全做出来。先做最简单的版本：

```
第 1 版：城市页 + "List Your Studio Free" CTA（已有 ✅）
第 2 版：认领按钮 + 免费注册
第 3 版：认领后管理后台
第 4 版：Pro 升级接预约
```

这样每步都有价值，每步都在推进。
