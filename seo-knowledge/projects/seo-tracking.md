## 排查清单（没索引/没排名时）

### 索引问题
- [ ] GSC → URL 检查 → 请求索引
- [ ] robots.txt 有没有屏蔽？
- [ ] 页面有没有 noindex 标签？
- [ ] 页面是否有足够的内链指向它？
- [ ] sitemap 是否包含该页面？

### 排名问题
- [ ] 目标关键词 KD 是否太高？（对比页 KD 10-20 合理，>25 可能太高）
- [ ] 竞对是否有我们没有的内容模块？
- [ ] 竞对是否有我们没有的外链？
- [ ] 页面内容深度够不够？（对比页 ≤1500字需要加）
- [ ] 页面是否有独特的"信息增益"？（竞品没有的数据/案例/角度）
- [ ] 内链够不够？（每页至少 3-5 条）
