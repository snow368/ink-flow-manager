## 截图规范

| 要求 | 说明 |
|------|------|
| **格式** | PNG 原图给我，我转 WebP |
| **尺寸** | 浏览器 1440×900 窗口截取 |
| **数据** | 用真实数据，不要空白状态 |
| **清理** | 不要暴露真实客户信息 |
| **命名** | 按上表文件名保存 |
