# Google 算法更新追踪

> 记录已知的 Google 算法更新及其对 SEO 策略的影响。

## 核心算法更新（截至 2026）

### Helpful Content System (HCS)
- **启动时间**: 2022年9月（全面升级）
- **影响**: 低质量、以 SEO 为导向而非为用户创作的内容排名下降
- **正面信号**: E-E-A-T（经验、专业性、权威性、可信度）
- **对 SaaS 的影响**: 博客内容必须先有价值，再谈 SEO

### Page Experience Update
- **核心指标**: Core Web Vitals (LCP, FID/INP, CLS)
- **INP 替代 FID**: 2022年7月起用 Interaction to Next Paint 替代 First Input Delay
- **对 Astro 的影响**: Astro 天然 CWV 优秀，这是优势

### Spam Updates
- **Site-wide Ads / Parked Domain**: 2023年2月
- **Expired Domain Rebranding**: 2023年5月
- **Fake Reviews**: 2023年10月
- **AI Spam**: 2024年初持续强化

### AI Overviews (AIO) / SGE
- **影响**: 搜索结果的 AI 生成摘要改变了点击模式
- **机会**: 内容结构要直接回答问题，争取被 AI 引用
- **风险**: 简单事实型查询的 CTR 下降

### Reviews Update (2024)
- **重点**: 虚假评论、未经验证的产品评论
- **对 SaaS 的影响**: 如果有用户评价页，必须真实可验证

## 持续追踪的信号

### E-E-A-T 强信号
1. 作者有实际行业经验（tattoo studio 运营背景）
2. 内容有具体数据、案例、截图
3. 关于页面展示真实团队
4. 引用权威来源

### Content Quality 信号
1. 原创性 > 汇编性
2. 内容深度 > 广度
3. 用户停留时间 > 跳出率
4. 分享/引用次数
