# 排名因子

> Google 搜索排名中已证实和推测的排名因子。

## 已证实的核心排名因子

### 1. 内容质量 (最高权重)
- 原创性
- 深度和完整性
- E-E-A-T 信号
- 用户满意度（停留时间、跳出率）

### 2. 反向链接 (高权重)
- 数量（域名多样性 > 总数量）
- 质量（DA/DR）
- 相关性（同行业链接价值更高）
- 自然度（不要买链接）

### 3. 技术 SEO (基础门槛)
- 页面加载速度 (Core Web Vitals)
- 移动端适配
- HTTPS
- 可抓取性 (robots.txt, sitemap)
- 结构化数据

### 4. 页面体验
- Core Web Vitals (LCP < 2.5s, INP < 200ms, CLS < 0.1)
- 无 intrusive interstitials
- 安全的浏览环境

### 5. 搜索意图匹配
- 页面内容与搜索 query 的意图一致
- 内容格式符合用户预期（信息型→文章，交易型→对比/产品页）

### 6. 域名/品牌信号
- 品牌搜索量
- 品牌词提及（不一定是链接）
- 品牌权威性

### AEO 内容优先于传统 SEO（2026 新趋势）
- Search Engine Land: "Why so much SEO work no longer drives growth"
- AI Overview 替代传统点击，策略转向"抢引用"优先于"抢排名"
- SaaS 最佳内容: 对比页、行业数据报告、决策指南

## SaaS SEO 特有因子

### Product-Led SEO
- 免费工具/计算器页面带来的自然流量
- 模板/资源页的引链接能力
- 对比页（alternatives page）的排名能力

### Content-to-Product 漏斗
- 博客 → 功能页 → 定价页 → 试用注册
- 内部链接密度和路径设计
- 每篇内容的 CTA 转化率

## 对 InkFlow 的策略意义

1. **内容优先**: 博客必须有真实 tattoo 行业洞察，不能是泛泛的 "best tattoo software" 灌水
2. **反向链接策略**: 通过免费工具、模板、行业报告获取自然链接
3. **技术优势**: Astro 框架天然 CWV 优秀，利用这个优势
4. **E-E-A-T**: 强调创始人/团队的 tattoo 行业经验
5. **Product-Led**: 考虑做一个免费 tattoo booking calculator 或 similar tool
