# 自动摇头吹风机支架 - 产品思路记录

## 产品定义
固定支架 + 电机摇头机构，夹住吹风机后自动左右/上下扫描
- 速度可调（快扫/慢扫）
- 高度可调
- 目标用户：长发女性、行动不便者、宠物店

## 技术验证
- 原理：步进电机 + 控制板 + 齿轮组 → 技术成熟
- 供应链：深圳/东莞有公模方案
- 关键元器件：步进电机¥5-15 / 控制板¥3 / 齿轮¥5
- 时间线：打样2周、开模4周、量产6周
- 结论：✅ 技术可行性高
