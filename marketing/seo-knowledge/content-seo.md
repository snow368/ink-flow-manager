# 内容策略原则

> SEO 内容创作的核心原则和最佳实践。

## 内容类型金字塔

```
                    ┌─────────┐
                   │  Pillar │  ← 核心页面（首页、功能页、定价页）
                  │  Content│
                 ┌───────────┐
                 │ Cluster   │  ← 主题集群（深度文章、指南）
                 │  Content  │
                ┌─────────────┐
                │  Support    │  ← 辅助内容（FAQ、对比、博客）
                │  Content    │
               ─────────────────
```

## 内容创作原则

### 1. 先价值，后 SEO
- 第一目标：解决用户真实问题
- 第二目标：通过 SEO 让内容被发现
- 永远不要为了塞关键词而牺牲可读性

### 2. 深度 > 广度
- 一篇 3000 字的高质量文章 > 三篇 1000 字的泛泛文章
- 覆盖子问题的方方面面
- 用真实案例、截图、数据支撑

### 3. 差异化角度
- 不要重复竞品说过的话
- 找到独特的视角（tattoo 行业特有洞察）
- 用自己的经验说话（"我们帮 500+ tattoo 工作室管理..."）

### 4. 结构化内容
- 短段落（2-4 句）
- 大量子标题（H2/H3）
- 列表、表格、对比图
- 关键结论前置（第一段就给出答案）

### 5. AEO 导向内容
- 每个 H2 标题就是一个可能的问题
- 答案直接在 H2 下给出，不要铺垫
- FAQ section 专门针对 "what is", "how to", "best" 类 query

## SaaS 内容类型清单

| 类型 | 目的 | 示例 |
|------|------|------|
| 对比页 | 捕获竞品流量 | "InkFlow vs Booksy vs Tattoo Smart" |
| 替代页 | 捕获迁移搜索 | "Booksy alternatives for tattoo studios" |
| 指南/教程 | 建立权威性 | "How to manage your tattoo studio in 2026" |
| FAQ | AEO 优化 | "Do tattoo artists need booking software?" |
| 行业报告 | 获取反向链接 | "2026 Tattoo Studio Technology Report" |
| 模板/工具 | 产品导流 | "Free tattoo contract template" |
| 案例研究 | 社会证明 | "How Studio X increased bookings 40%" |

## Tone of Voice
- 专业但不枯燥
- 展示行业知识（不是泛泛的 SaaS 公司）
- 真诚，不夸大
- 避免过度营销语气
