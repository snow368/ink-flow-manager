# 网站类型 SEO 策略分类

## 适用分类
商业策略, 网站类型

## 1. SaaS（订阅制软件）

**代表：** InkFlow
**目标：** 获取付费订阅用户
**转化路径：** 搜索 → Landing Page → 注册试用 → 付费

### 策略
- **关键词：** 功能词、对比词、替代品词、痛点词
- **内容格式：** 对比页（转化最高）、功能页、替代品页、博客
- **外链：** G2/Capterra、集成伙伴目录、SaaS 目录
- **KD 目标：** < 20（初期），20-40（中期有外链后）
- **量级：** 20-50 页精品内容即可起步
- **核心 KPI：** 注册转化率、付费转化率

---

## 2. B2C 卖货（电商 / DTC）

**代表：** —
**目标：** 直接销售产品
**转化路径：** 搜索 → 产品页 → 加购 → 下单

### 策略
- **关键词：** 产品词、品牌词、"best X"、"X vs Y"、"buy X"
- **内容格式：** 产品页、分类页、购买指南、评测对比
- **外链：** 评测站、导购站、Influencer、社媒
- **KD 目标：** 产品词可以接受中高 KD（因为搜索意图精准）
- **量级：** 产品数量决定了页面数量
- **核心 KPI：** 加购率、下单率、ROAS

---

## 3. B2B 供应商（批发 / 代工）

**代表：** —
**目标：** 获取询盘/订单
**转化路径：** 搜索 → 产品目录 → 询盘 → 订单

### 策略
- **关键词：** 长尾产品词、规格词、"X manufacturer"、"X supplier"
- **内容格式：** 产品详情、规格表、资质认证、工厂介绍
- **外链：** 行业目录、B2B 平台（Alibaba/Made-in-China）、行业协会
- **KD 目标：** 长尾词 KD 天然低，容易做
- **量级：** SKU 量决定
- **核心 KPI：** 询盘量

---

## 4. 内容站（广告 / Affiliate）

**代表：** hezhiyan7 模式、图片识别站
**目标：** 流量变现
**转化路径：** 搜索 → 内容页 → 广告点击 / Affiliate 链接

### 策略
- **关键词：** 低 KD 热词、信息型长尾
- **内容格式：** 批量生成、聚合页、模板页
- **外链：** 不重要，靠量取胜
- **KD 目标：** < 10，越低越好
- **量级：** 几百到几百万页
- **核心 KPI：** PV、广告点击率、Affiliate 转化

### 注意
- Google 2026 更新在杀纯模板内容站
- 需要每页有真实价值，不能只换关键词

### 快速起量案例

来源：@hridoyreh（Hridoy Rehman）2026/06/07

新站从 0 到流量增长的案例：

| 指标 | 数值 |
|------|------|
| 建站时间 | 2月1日 |
| 总流量 | 1,700+ |
| Google 展示量 | 65,200+ |
| 平均 CTR | 2.7% |
| 近期趋势 | 流量因 Google 更新上涨 |
| 目标 | 500 次/天 |

**启示：**
- 内容站新站确实可以在 Google 更新中获得流量提升
- 65k 展示量仅换来 1.7k 流量 → CTR 2.7%，说明 title 有优化空间（还有提升余地）
- 核心：低 KD 关键词 + 批量内容覆盖 → Google 大量展示

## 5. 品牌站（品牌认知）

**代表：** —
**目标：** 品牌声量 + 直接流量
**转化路径：** 品牌搜索 → 官网 → 了解产品/服务

### 策略
- **关键词：** 品牌名、行业思想领导力话题
- **内容格式：** 品牌故事、原创研究、案例、PR
- **外链：** 媒体报道、PR、合作伙伴
- **KD 目标：** 无所谓，品牌词没有竞争
- **量级：** 少而精

---

## 跨类型通用原则

1. **先做 BOFU（底部漏斗）内容** — 对比页、替代品页，转化最高
2. **再补 TOFU（顶部漏斗）内容** — 博客、指南
3. **外链质量 > 数量** — 10 个 DA 60+ 外链 > 100 个 DA 20 外链
4. **内容质量 > 内容数量** — Google 2026 更新惩罚低质内容
5. **同一意图关键词要分组优化** — 不要零散做
