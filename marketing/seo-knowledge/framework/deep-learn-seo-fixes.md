# 深度拆解：fba（Flavio Amiel）SEO 实操

## 适用分类
技术SEO, 修复方案

## 策略 1：5 SEO Fixes Before Dinner

**适用场景：** 任何网站，每季度跑一轮

### 执行步骤
```
Step 1: 打开GSC → 按点击量排序前10页
Step 2: 每页加3条内链（从站内其他相关页面链过来）
Step 3: Title改成精确匹配目标关键词
Step 4: 底部加FAQ block（3-5个GEO优化问答）
Step 5: 把超过200KB的图片压缩
时间：一顿饭的功夫
```

## 策略 2：11-20 排名优化法

**适用场景：** 已有排名的页面，差几步进首页

### 执行步骤
```
Step 1: GSC → 按位置 11-20 筛选
Step 2: 按展示量降序排序
Step 3: 挑前5个
Step 4: 每个页面做3件事：
  a. 把H1改成精确匹配目标查询词
  b. 加5条从其他页面链到这页的内链
  c. 扩展真正回答问题的那一段
效果：2-3周后上首页
```
