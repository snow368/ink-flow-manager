## 适用分类

## 核心案例：B2C 品类页 SEO 突破

### 做了什么

**Step 1：优化 Collection 页面体系**
- 优化每一个 collection 页面
- 建立 sub-collections（子品类页）
- 同类 collection 之间做 internal link

**Step 2：建外链**
- 80/20 原则：80% 指向首页，20% 指向目标 collection 页

**结果：增长缓慢。**

### 转折点：补充 Blog 内容

然后针对每个品类发布了支撑性博客：

> 每个品类 5 种 blog 类型：
> - **竞品对比**（Comparisons）
> - **购买指南**（Buying Guides）
> - **常见问题**（FAQs）
> - **安装/维护指南**（How to install, maintain, etc.）
> - **按场景推荐**（Best for X — "best for back pain", "best for stomach sleepers"）

**30 天后** → 部分高竞争关键词进入 **前 3 名**。

## 核心洞察

### Collection 页本身不够

> Optimize collection pages + sub-collections + internal link + backlinks = 增长缓慢

**问题**：Collection 页商业意图强但缺乏内容深度，Google 很难判断其权威性。

### Blog 内容起了杠杆作用

> 针对每个品类发 5 类支持性博客 → 30 天进前 3

**机制**：
1. Blog 覆盖了用户的信息类搜索意图（buying guide、comparison、FAQ）
2. Blog 增加了站内 topical authority（品类关键词的语义深度）
3. Blog 之间的 internal link + Blog → Collection 的链接传递了权重
4. 整个品类关键词集群在 Google 眼中变得更专业

### 和 hezhiyan7 策略的对比

| 维度 | hezhiyan7（内容站） | KaiCromwell（B2C） |
|------|-------------------|-------------------|
| 网站类型 | 工具站 / 内容站 | Shopify 电商 |
| 核心策略 | 低 KD 词 → 海量内页 | 品类页优化 + Blog 支撑 |
| 转化方式 | AdSense / Affiliate | 直接卖货 |
| 内容量 | 几百到几百万页 | 每个品类 5 篇支撑博客 |
| 见效时间 | 长期积累 | 30 天可见效果 |

## 对 InkFlow 的启发

InkFlow 是 SaaS，不是电商，但同样有"品类页 → Blog 支撑"的逻辑。

### 映射到 InkFlow 的页面结构

```
InkFlow 品类页（= Collection 页）：
  - 功能页（Booking / CRM / POS / Waiver / Aftercare...）
  - 对比页（vs TSP / vs Tattoogenda / vs Booksy...）
  - 城市页（/book/[city]）

InkFlow 支撑博客（= KaiCromwell 的 5 类 blog）：
  每个品类需要覆盖：
  1. 对比文 → 我们有：vs TSP, vs Tattoogenda, vs Booksy, vs Vagaro
  2. 指南文 → 需要："How to choose tattoo studio software"
  3. FAQ → 需要：每个功能的 FAQ 页面
  4. 按场景推荐 → 需要："Best tattoo software for solo artists"
                          "Best tattoo software for multi-artist studios"
                          "Best tattoo software for walk-in shops"
  5. 教程/维护 → 需要："How to set up online booking for tattoo shops"
                        "How to digitize tattoo consent forms"
```

### 已完成的 blog（对标 KaiCromwell 模型）

| Blog 类型 | 已完成？ | 建议 |
|-----------|---------|------|
| 竞品对比 ✅ | 4 篇（TSP/Tattoogenda/Booksy/Vagaro） | — |
| 购买指南 ⚠️ | 有 Pricing 页但无完整 Buying Guide | 新写 "How to Choose Tattoo Studio Software in 2024" |
| FAQ ⚠️ | 功能页有散落 FAQ | 整合为独立 FAQ 页面集群 |
| 按场景推荐 ❌ | 没有 | 新增 "Best for Solo Artists / Multi-Artist / Walk-in" |
| 教程/指南 ❌ | 没有 | 新增 setup / migration 教程文 |
