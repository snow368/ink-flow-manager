### 4 个角色模板

| 角色 | 触发 | 输出 |
|------|------|------|
| Content Creator | 每天早上 7am | 趋势扫描 → 5 个内容 ideas |
| Freelancer | 新询价 | 线索评分 → Call or Skip |
| Researcher | 持续 | 网页监控 → 过滤 → 每日摘要 |
| Everyone | 每天早上 | 日历 + 邮件 + 网页 → 一条晨间简报 |

### 3 条工作流

```
Web Scan → 5 Ideas Delivered
New Inquiry → AI Scoring → Call or Skip  
Web Monitor → Filter → Daily Digest
Calendar + Email + Web → Claude → One Message
```

## 二、对 SEO 知识库/内容生产的应用

这个框架可以直接用在 SEO 工作流中：

| 用途 | 触发 | Claude 做什么 |
|------|------|-------------|
| 竞品监控 | 每天早 | 扫竞品新内容 → 摘要 → 找内容缺口 |
| GSC 分析 | 每周 | 拉 GSC 数据 → AI 分析 → 输出 Quick Wins |
| SEO 实验日志 | 每改动 | 自动记录 annotation → 2 周后拉数据 |
| 内容生产 | 每 idea | 研究话题 → 写 brief → 输出初稿 |

**核心逻辑：** 不是问 Claude 单个问题，而是设好 trigger + workflow，让 Claude 持续自动运行。
