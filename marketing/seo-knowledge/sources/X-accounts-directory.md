## 外链机会类型说明

X 上常见的免费外链机会：

| 帖子类型 | 说明 | 能用吗 |
|---------|------|-------|
| "Drop your link" | 互推帖 | ✅ 有用但权重低 |
| "Help a founder" | B2B SaaS 互帮帖 | ✅ 可能有真实转化 |
| "SaaS directories" | SaaS 目录站 | ✅ 高价值 |
| "Guest post requests" | 博客征集投稿 | ✅ 高质量 |
| "HARO / featured" | 记者征集专家评论 | ✅ 质量极高 |
| "Broken link building" | 外链机会分享 | ⚠️ 耗时间 |

## 付费 vs 免费外链说明

**Adsy 这类平台：** 本质是买 Guest Post，不是买链接。你付钱让人家发你的文章到他们博客上，文章里自然带你的链接。价格 $25-150/篇不等。

**和 PBN（垃圾外链）的区别：**
- ✅ Adsy → 真实博客、真实流量、真实文章，Google 认可
- ❌ PBN → 专门做外链的垃圾站群，Google 会罚

**你的立场：** 你说过不想花钱。那前期先走免费路线（G2/Capterra/Trustpilot/集成目录），后面如果有预算了 Adsy 可以作为补充。

## 用法

1. 发现新账号 → 加到目录，标注类型和状态
2. 学习内容 → 在 `D:\seo-knowledge\sources\` 下建分析文件
3. 判断适用性 → 按网站类型打标签（SaaS / B2C / B2B / Content / Brand）
4. 定期回顾 → 冲突点对比 → 调整策略
