# Tan SEO 203 — 旧页面翻新（不写新内容提流量）

## 核心公式（诊断前提）
> 流量 = 排名 × CTR × 搜索量

内容质量**只影响排名**（1/3 战场）。流量掉了先诊断是哪个因子掉，再下药。盲目重写内容常是对着错误问题。

## 翻新三件事（都不写新内容，按见效/成本排序）

### 1. 改标题 + Meta Description（最快、最低成本，单页 15 分钟）
- **找目标页**：GSC → 效果 → 搜索结果 → 页面 → 按展示次数排序 → 筛选 **CTR < 3%**（展示高但没人点）。
- **标题改写逻辑**（三步）：① 说清是什么 ② 放 1-2 个买家关心参数 ③ ≤ 55 字符。
  - 反例：`Industrial Pump Manufacturer - China Supplier`（只堆词，无承诺）
  - 正例：`Chemical-Resistant Pump | 316L SS, pH 1–14, -30~180°C`
- **Meta Description 当广告写**：解决什么问题 + 具体卖点 + 行动号召（CTA）。
- 不写任何新内容。

### 2. 内链重新指向（30-45 分钟，激活已有权重）
- 外链是借的权重，**内链是自己手里已有的**、完全可控，最易被忽视。
- 旧页流量下滑有时只是「强页面从未链过它」。
- 操作：GSC → 页面 → 按点击量排序取**前 10 权重大户** → 从相关高流量页在自然位置插入指向目标页的链接。
- **目标：每个要救的页加 3-5 条内链**；锚文本混用自然变体（别千篇一律）。
- 加完在 GSC「URL 检查」手动请求重新抓取。

### 3. 合并同类页 + 301 重定向（1-2 小时，4-8 周见效）
- 同一类产品/主题建了 2-3 个相似页 → **关键词蚕食（Keyword Cannibalization）**：权重稀释，Google 不知推哪个，结果都没推好。
- **判断**：GSC → 搜索词栏输入核心词 → 切「网页」标签 → 同一词对应 2+ 个 URL 即蚕食。
- **处理**：合并内容到一页，旧 URL 301 到新 URL（权重大部分转移）。Shopify：导航 → URL 重定向。
- 合并后：① 所有指向旧页的内链更新为新 URL ② GSC 提交新 URL 请求索引。

## 什么情况才真需要重写内容（信号）
三件事做完等 6-8 周还没回升，才考虑动内容。重写信号：
1. 内容有过期事实（价格/参数/认证已变）
2. 搜索意图变了（原「了解产品」→ 现「找供应商比价」）
3. 排名停 6-10 且竞对深度明显更高
4. 页面不足 400 字（本质存根）

## 翻新顺序（核心心法）
> **诊断原因 → 最小有效动作 → 等待观察 → 必要时才动内容**

大多数人跳过前三步，直接做最重最慢最没必要的重写。
