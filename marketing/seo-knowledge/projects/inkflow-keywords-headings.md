# InkFlow 关键词 → 标题布局规划

按 DR 阶段分配关键词到 H1-H3，来源：关键词矩阵 `inkflow-keywords.md`

---

## 一、核心原则

```
H1: 主关键词（页面核心意图，KD 决定什么阶段做）
H2: 相关长尾 + 子话题
H3: 具体功能点 + 细分场景
P/UL: 自然包含同意图的其他关键词
```

**同一个意图的关键词组放在同一个页面**，不做独立页面浪费资源。

## 七、怎么确认关键词有没有排上

域名下来后，每篇内容发布后：
1. Google Search Console 看该 URL 被索引了哪些词
2. 手动搜目标短语看排在第几页
3. 如果 4 周后还没收录，检查是否是索引问题（GSC → 请求索引）
